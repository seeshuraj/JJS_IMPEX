# JJS IMPEX

A professional website for JJS IMPEX showcasing products and services.

## Features

- Modern, responsive design
- Interactive animations and scroll effects
- Professional product showcase
- Contact form with glassmorphism effects
- Premium typography and visual effects

## Technologies Used

- HTML5
- CSS3 (with animations, glassmorphism, and modern effects)
- Vanilla JavaScript
- Google Fonts (Inter, Poppins)

## Local Development

Simply open `index.html` in your browser to view the website.

## Contact

For inquiries about JJS IMPEX products and services, please use the contact form on the website.
